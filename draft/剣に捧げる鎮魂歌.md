# 剣に捧げる鎮魂歌

※書きかけです
```
アフターオレオールの話。
魔族と戦って死した戦士達の剣塚。そこには戦士達が使っていた武器と防具が遺骨代わりに眠っている。
だが、フェルンは知っている。一本の剣だけは塚に刺さずにずっと手入れして持ち続けていることを。
兄シュトルツの墓は既にあることは知っている。ではあれは……？

シュタルクとフェルンの第一子が生まれて少し経ったある日、シュタルクの師であり父親代わりのアイゼンがが2人に告げる。

「剣塚の代わりになるいい霊石が見つかった。墓を作り直し、今度こそ彼らを弔おう」

しかし、フェルンは知っている。ただ一人、シュタルクが弔えずにいる人がいることを。
シュタルクは、夫は、誰も遣わぬ剣を前に一体何を思うのか……
```
