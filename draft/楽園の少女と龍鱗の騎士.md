# 楽園の少女と龍鱗の騎士  

```  
オリジナル作品  

地上に降り立った24匹の「竜」のもたらした様々な技術により高度に発展した旧世代の文明が行方不明担ってから数百年。  
人以外に破格の力を与えるドラゴンスケイルというウイルスがはびこり、世界は硬い甲殻と鱗を持つ「竜」と呼ばれる生物の特性をもつ「攻殻獣」で溢れかえる世界。  

「攻殻獣」の跋扈により何もかも失った人類はゼロからリスタートして独自の文明の再形成を余儀なくされた人類。  
栄華を誇った古代文明の残した「天空都市エデン」の天上人。地上世界に暮らす地上人に分かれて生活している世界観。  

エデンに生きている人類は世界と隔離し状況は不明。地上は一部の組織化した国家と、集団としている組織で「攻殻獣」と競り合っている。  

竜騎士という「攻殻獣」と戦うための特殊な技術を持つ者たちが存在し、彼らは狩った「攻殻獣」の素材を利用した武器を使った武具で生活している。  

また、一部の人間はエデンからもたらされる不可思議な力を操る「亜法術」を使うことが出来る人間も存在し、特別視されている。  

「竜」とは「攻殻獣」の元凶とされているが、実際には謎に包まれており、一部では神の使いとも言われている。  

非常に強力な「攻殻獣」が守る人類未踏は領域に存在する「楽園」と呼ばれる場所。そこには伝説の天空都市につながる塔があるとされている。  

```  

## 序章：楽園の少女と竜  

『緊急事態発生！緊急事態発生！ドラゴン型の潜入を感知。捕獲、または掃討のプログラムの開始してください』  

「お母さん。騒ぎになっちゃった」  
そう呟くのは、青白いロングヘアを風に揺らす少女の言葉。  
彼女は翼を生やした2本の角……一本は折れているが……そんな異形の生物の背に乗る。  

『面倒じゃのう……、さっさと逃げるかか。ウヌが居るならもうここには用はない』  

異形の生物は口を開き、少女に答える。人の分かる言葉を話すそれに少女は疑問を覚えない。  

「ところで、お母さんは、どうして追われているの？」  
『いや、まあ、こんな見た目だからの。あとまあ、お前をさらったからじゃろうな』  

少女は異形の生物の言葉に首を傾げる。  

「私を？何で？」  
『お前が妾に必要だからじゃ。お主もあの場所であれに付き合い続けたかったのか？』  
「判んない……」  

そんなやり取りの中、下方から声がする。  

「攻撃性亜法、一斉掃射ぁぁ！」  
『あー、もう面倒くさい！！ 契約がなければ薙ぎ払ってやろうものを！！！』  

異形のそれは怒りの声を上げると、少女を背に乗せたまま急上昇する。  
「お母さん。急に動かれると落ちる」  
『しっかり捕まっとれ！』  

全て回避した異形の生物はやれやれとため息を付いた。  

『飛行部隊が出てきそうか……捕まっとれ。全力で飛ぶぞ』  
「うんわかった」  
『そう言えば、お前名前は？』  
「知らない。9番って呼ばれていいた。お母さんはなんでお母さんなの？」  
『番号は味気ないの。……ふむ。ノルンだ』  
「なにそれ？」  
『お前の名前じゃ。妾が名付けた。9番で思いつきじゃが……女神を意味する名前でもある』  
「そうなんだ。わかった。私はノルン。で、お母さんは？」  
『むふ。それはな–―』  

そんなチグハグな会話を交わす異形の生物と少女。  

「駄目です！ 逃げられます！追いつけません」  
「クソ！周辺の探索を怠るな！必ず捕まえろ！」  

彼女たちを追う者たちは施設中の明かりを煌々と光らせ臨戦事態の立ち上げる。  
周辺の領土にも緊急指令を出したが、少女を載せた異形を見つけることは出来なかった。  


## Section 1：砂漠を這う竜と行き止まり  

「誕ちゃん、すまんな……今は船を出せんのだよ。砂漠竜が暴れてるからさ」  
「そうですか……」  
「わりぃ……もうすぐ休眠期なはずなんだよ」  

少女が交渉しているのは砂漠を進む民間の船の主の男。  
大陸の南部、広大な砂漠が広がる地域。ここでは砂上を進む船を利用して移動することが一般的だ。  

「ちなみに、休眠期というのは後どれぐらいで？」  
「……すまん。分からん。実はそろそろ来ていてもおかしくない頃合いで……我々も困っている。  
　昔っから世話になってる外れに住んでる竜騎士様が動いてくれりゃな」  

流通の拠点となる大きな街。ひとまず生活には困らないものの……  

「そうですか……」  
「わりいな……」  

船の主の男は少女に謝罪する。  

「いえ、いいんです。私も砂漠竜に襲われたくはないので」  

船の主から別れた少女はため息をついた。  

「砂漠竜か……。本来、お母さんの仲間だよね？」  

誰に言うでもない言葉に、彼女の大きなカバンの中から返事があった。  
声だけで言うなら幼女の様な声。  

「そうじゃな。下層の一匹だ。一声あれば大人しくなるようなやつだな」  

砂漠竜という名前だが、実際は巨大な口のついた大きなミミズの化け物。  
砂漠の砂を掘り進み。砂上を這うように移動し。動くもの全て丸呑みする厄介な生物だ。  

背中から聞こえる快活に笑う声に少女はため息をつく。  

「じゃあ、お母さんが一声かけたら私達大儲けだよ」  
「それは無理な話じゃ。なんせもう言うことを聞かん」  
「だめじゃん……」  
「せやの……」  

$1  の街に辿り着いてから数日。そんな感じでこの先の道を進めないでいる状況にいる。  

「お母さんが翔べたら楽なのに」  
「無理じゃのー。妾が元の姿に戻るのはあと100年かかる」  
「とりあず。私が生きている間は無理ってことね」  

わざとらしい咳払いが聞こえる。  

「すまんのう。妾のせいで」  
「いいよ。お母さんが悪いわけじゃないから」  
「そう言ってくれると助かる」  

少女は手に持っていた杖をクルッと回して両手で持ち直す。  

「あんまり気が進まないけど……退治するしか無いか」  
「そうじゃな。アレの肉はそこそこ美味い」  

見た目があれなので、アレの肉……と少女は嫌そうな顔をする。  

「早速行くぞ！ノルン！」  

少女の背中の大きなバックから顔を出したのは、緑の髪の童女。  

「わかりました。お母さん」  

それに応じた少女、ノルン。彼女は空のように青白いロングヘアをなびかせて走り出した。  

✧　✧　✧　✧  

まあ、さりとて。戦力不足は否めない。  

「巨大化した砂漠竜か……まあ、今の戦力なら五分五分じゃな」  
「お母さん、本気で言ってる？  
　船ごと丸呑みされるよ」  

巨大さとはある種の戦力だ。攻撃力がなくても巨大であれば生物的に強靭である。  
そして砂漠竜は巨大な上に攻撃的だ。  

「まあ、丸呑みされても割となんとかなる」  
「嫌なんですけど……」  

お母さんと呼ばれた童女はフームと考える。  

「この街にいるという竜騎士。そいつを味方につけるか」  
「この街に来てから、時々名前きくよね。どうして、そんな人がいるのにこんな状況なの？」  

緑髪の童女はぷいっと顔を背けた。  

「さあの。ただこの地方の竜騎士には覚えがある。  
　面倒くさいやつじゃ」  
「お母さんその人と会った事あるの？」  
「昔、喧嘩をふっかけられた。妾の美しい翼と角に傷をつけよって」  

童女の姿に角も翼もあるわけがないが……ノルンはうーんと唸る。  

「あ、お母さんの片方の角、その人に切られたの？」  
「ふん……」  
「すごい人なのね」  
「まあ、気に食わんがな……あの実力なら砂漠竜も倒せるじゃろう」  

彼女がそう言うならそうなのだろう。ノルンは納得するが一つ気がかりなことがある。  

「お母さん、それ何年前の話？」  
「あー、うーん、ついこないだの話しじゃ。ほれ、50年ほど前じゃな」  

何の気なしに言う。現在の大陸共通の新暦は15000年。  
彼女はそれよりも前から生きている……らしい。  

「お母さん……多分もう生きてないよ。その人」  
「そうかのう……」  

ダメ元。致し方ない。とにかく街の人の言う竜騎士様を探すことにした。  

## Section 2：はぐれの竜騎士  

竜騎士とは、現在世界中で人類の天敵とも言える「攻殻獣」と呼ばれる生物と戦うための特殊な技術を持つ者たちのことだ。  
彼らは、機械式竜撃槍と呼ばれる槍を用いて戦う、武装兵士だ。  

もちろん、攻殻獣と戦うのは竜騎士だけではない。武装戦車や、空戦機、そして亜法術を使う者たちもいる。  
『竜騎士とはその中で、あえての白兵戦を選んだ気の狂った連中じゃ』というのはお母さんの説明。  

「お母さん。そんな気の狂った戦い方なのに、何で正規の職業になってるの？」  
「まあ、そりゃ、案外攻殻獣に対して有効だったからじゃ。正確に言うと機械式竜撃槍という頭のおかしい武装が案外うまくいった」  
「そうなんだ……。見たこと無いから知らないんだけど」  
「攻殻は一般的な銃火器ではびくともせん。装甲が厚いからのう。  
　戦車や空戦機の攻撃も火力次第じゃ。そんな中、連中は取り付いて内部破壊を効率的に狙う武装を開発した」  
　
機械式竜撃槍は……名前の通り機械式の槍である。特徴としては、柄の末端にブースター射出口がある事。そして、槍の先端に砲身がついている事。  
攻殻獣は空を飛ぶ物も多い。そのため、槍のブースターで一気に飛び上がり、その勢いで攻殻獣に突き刺す。  
その状態から、槍の先端にある砲身で攻殻獣の内部から撃ち抜く。  

「という、頭のおかしな武器じゃ。割と痛い」  
「痛いで済むんだ……」  
「まあ、小型や中型の連中なら一撃で爆散するのう。妾は……無敵のじゃもの」  

フフン、と自慢げに言うのでノルンはため息をついた。  

「お母さん、今実質ほぼ幼女でしょ」  
「む……痛いところを付くな。でもまあ、もうちょっと強くはなれるぞ。5分ぐらい」  
「短いよ……お母さん」  

✧　✧　✧　✧  

――キィィィィ  

高音の鳴き声とも羽音ともつかない音を立てるのは一見は蟲の様な攻殻獣。  
岩場に座ったままの人間。腹をすかせたその攻殻獣は勢いをツケてそれを狙う。  

「……お前が今日の晩飯だ」  

そう言った黒髪の青年は足元に隠していた槍に手をかける。  
槍の先端には大きなアンカーと砲身。極端に肥大化した護拳にはブースターの射出口がついている。  

―― 機械式竜撃槍  

彼の愛槍。  

此方に向かって跳んでくる攻殻獣に向かって彼は走り出す。  
自慢のブースターは……点火しないらしい。  

「これで十分だあああああ」  

脚力で駆け出した青年は、停止する様子のない攻殻獣と激突し、その胴に槍の先端を射し込む。  
攻殻の隙間。ここなら無理なく刺さる。  

――ぴぃぃぃぃぃ  

妙な鳴き声は断末魔か。攻殻獣は腹を刺し貫かれて苦しんでいるようだ。  
人の２倍はありそうな攻殻獣。しかし、それは青年の刺突に吹き飛ばされた。  
青年の素の脚力なのか槍に刺さったまま岩壁に攻殻獣を打ち付けた。  

「火薬は使えねえからな。こっちで十分だろ！」  

青年は槍の柄を、グリップで握り回転させる。  

「撃ち抜けぇ！！」  
と言いつつ砲撃は打たない。なぜなら爆発四散するから。  
あと、火薬ももったいない。  

ズンッ！！という振動恩と共に背面の殻が砕け、刺突した槍の先端……ではなく。取り付けていたアンカーの杭が飛び出した。  

どう形容したら良いのかわからない断末魔の声を上げ。  
その蟲の様な攻殻獣は、岩壁に打ち付けられたまま動かなくなった。  

✧　✧　✧　✧  

「ふぅ……」  
青年は息を吐くと、槍を引き抜く。  

「これで……晩飯は確保だな」  
青年はそう言いながら、槍を引き抜いた。そして腰からナイフを取り出す。  

「これが、一番面倒なんだよな……」  

晩飯と言ったが、青年は食べるつもりはない。  
彼は、攻殻獣の身体は様々な道具となる。硬質な部位は武器にもなるし、体液は燃料の素材にもなる。  

「脚と……、殻と、あと内蔵と……血液は燃料に使えるな」  

どういう構造か分からんが、コイツラの体液は発火する。  
そういう特性を活かして、攻撃してくる奴らも居る。  

「ようやくお前の機能も復活するぞ」  

青年はやりに向かってそう語りかける。そう。使わなかったのはそのせい。  
燃料が足りないのだ。  

「よし、全部畳んで村に売りさばきに行くか！！」  
青年はそう言うと、槍を肩に担ぎ、攻殻獣の残骸を袋に詰めて歩き出した。
