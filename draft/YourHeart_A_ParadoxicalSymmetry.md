# 君の心は二律背反シンメトリー  ～Your Heart: A Paradoxical Symmetry~　

```  
分裂の魔法。立ち寄ったダンジョン内で掛けられてしまった魔法によりフェルンが二人に分かれてしまう。  

「どっちが偽物って魔法じゃないね……多分フェルンが2つに別れちゃってる」  
「ええ……」  

という状況から始まる物語。  
同一人物と思いきや、陰陽が分裂しており。徐々に二人のフェルンの行動は変容していく。  
陽のフェルンは表情豊かに、優しく、何でも肯定してくれる。  
陰のフェルンは厳しく、小うるさく、嫉妬深い……  

そしてなぜか二人のフェルンはやたらにシュタルクによく構う。  

ある日、陰フェルンは自分と陽フェルンの差に気付き……その心歪みはやがて軋みを見せて――  
```  


## 二人のフェルン  

「シュタルク様、昼食のパンです。はいあーん」  

嬉しそうにフェルンが差し出してくるちぎられたパン。  
口元に差し出されたそれにためらいな横目で隣人を見たシュタルク。  
パーティーん指導者、魔法使いフリーレンは「うん」と頷いた。  

（他人事と思いやがって！！）  

「くっ……あーん」  
「はい」  

口を開けたシュタルクに喜んだフェルン。彼女は手に持ったパンのかけらを口に入れようとする。  
そんな刹那背中にどんとぶつかる何か。  

「シュタルク様！ パンばかりは体によくありません。  
　野菜を。野菜を食べてください。ビーンズも入っており健康にも良いものです」  
「お……おう。食べるよ。皿と、フォークくれる？」  

お皿を要求する手を見たフェルンはむぅと頬を膨らませる。  

「お口！開けてください！」  
「えぇ……」  
「はい、パンですよー」  

正面にはサラダと一緒に豆の刺さったフェルンの差し出すフォーク。  
と逆サイドからも口元に差し出されるパンの欠片を手に持ったフェルンの指。  

そう、現在シュタルクは二人のフェルンに挟まれている。
